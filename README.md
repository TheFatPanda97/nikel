### Self Hosting

1. git clone
```
git clone https://github.com/nikel-api/nikel.git
```
2. cd into nikel-parser submodule
```
cd nikel/nikel-parser
```
3. update nikel-parser submodule to latest
```
git pull origin master
```
4. cd into nikel-core
```
cd ../nikel-core
```
5. build nikel-core
```
go build
```
6. run nikel-core
```
Windows
./nikel-core.exe

Linux and macOS
./nikel-core
```
